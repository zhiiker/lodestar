    Private knownValidators¶

    • knownValidators: Map‹number, number› = new Map()

    Defined in packages/lodestar/src/network/attestationService.ts:63

    A collection of seen validators. These dictate how many random subnets we should be subscribed to. As these time out, we unsubscribe from the required random subnets and update our ENR. This is a map of validator index and its last active slot.

    Private pruneExpiredKnownValidators¶

    ▸ pruneExpiredKnownValidators(currentSlot: Slot): void

    Defined in packages/lodestar/src/network/attestationService.ts:213

    A known validator has not sent a subscription in a while. They are considered offline and the beacon node no longer needs to be subscribed to the allocated random subnets.

    We don't keep track of a specific validator to random subnet, rather the ratio of active validators to random subnets. So when a validator goes offline, we can simply remove the allocated amount of random subnets.

    Parameters:

    Name Type
    currentSlot Slot

    Returns: void
